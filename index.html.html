Универсал — система ЧПУ

Программа для управления станками с числовым программным управлением

О программе

Автор и правообладатель: Рудаков Александр Александрович

Программа разработана в Российской Федерации. Правообладатель — гражданин РФ. Исключительные права принадлежат Рудакову Александру Александровичу.

Программа "Универсал — система ЧПУ" предназначена для автоматизации управления станками с числовым программным управлением (ЧПУ). Она позволяет осуществлять контроль и управление процессами 3D-печати, фрезеровки и лазерной гравировки.

Область применения: производственные предприятия, мастерские, учебные заведения.

Основные функции

  • Управление станками с ЧПУ: поддержка режимов 3D-печати, фрезеровки, лазерной гравировки
  • Генерация G-кода: автоматическая генерация управляющих программ на основе 3D-моделей (STL-файлов)
  • Ручное управление осями: X, Y, Z, E, A, B, C
  • Визуализация процессов: 3D-визуализация моделей и траекторий инструментов, предварительный просмотр
  • Режим симуляции: тестирование G-кода без подключения к реальному оборудованию
  • Подключение к оборудованию: через COM-порты

Системные требования

  • Операционная система: Windows 7/8/10/11 (рекомендуется Windows 10/11, 64-бит)
  • Процессор: четырёхъядерный с тактовой частотой не менее 3.0 ГГц (например, Intel Core i5/i7 или аналогичный)
  • Оперативная память: 8 ГБ или более
  • Жесткий диск: SSD объёмом не менее 50 ГБ свободного пространства
  • Видеокарта: дискретная графика с поддержкой OpenGL 3.0 и выше (например, NVIDIA GeForce GTX 1050 или аналогичная)
  • Порты: наличие USB-портов для подключения оборудования, COM-порт (или адаптер USB-to-COM) для связи со станками с ЧПУ

Объём программы: 63.6 МБ

Язык программирования: Python

Используемые библиотеки: PyQt5, Matplotlib, NumPy, numpy-stl
